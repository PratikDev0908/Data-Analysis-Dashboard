This project is an interactive Human Resource Analytics Dashboard built using Tableau to analyze employee data and generate insights related to hiring, attrition, demographics, performance, and salary distribution.

##  Objective
- Track hiring and termination trends  
- Analyze employee demographics  
- Compare education level with performance  
- Study salary patterns across roles and age groups  
- View department and location-wise employee distribution

- Tableau Public

 ##  Dashboard Features
- KPI cards for Active Employees, Hired, and Terminated  
- Department-wise employee distribution charts  
- Gender ratio visualization  
- Education vs Age analysis  
- Education vs Performance matrix  
- Salary vs Age scatter plot  
- Location-based employee map  
- Interactive filters for deeper exploration
